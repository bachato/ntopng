.. _GrafanaIntegration:

Grafana
#######

ntopng-generated historical flows and alerts can be visualised in Grafana.

When using ClickHouse as the timeseries backend, ntopng provides a built-in Grafana
dashboard that can be downloaded directly from the ntopng Preferences page. See the
:ref:`BasicConceptsTimeseries` page for details on how to download and import the dashboard.

For additional Grafana dashboard examples and configuration templates, refer to the
`ntopng Grafana repository <https://github.com/ntop/ntopng/tree/dev/grafana>`_ on GitHub.
