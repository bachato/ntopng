RESTful API
===========

ntopng provides a RESTful API for exporting data through HTTP including 
Interfaces/Hosts/Flows information and raw PCAP data. nEdge, being based
on ntopng, also exposes those endpoints. Please refer to the ntopng 
documentation for further information.

In addition to the ntopng endpoint, nEdge extends the API with additional
endpoints which are documented here.
